# VGReborn（来自社区）

## 项目介绍

【VGReborn】是虚荣社区版的第三方游戏匹配平台，通过加速器+MITM技术自动绑定玩家状态到本平台账号系统，VGReborn将能够：

- 捕获玩家在线、接受比赛、拒绝比赛、游戏结束等状态, 通过游戏记录计算玩家排位等级；
- 看到在线人数，看到3v3/5v5等模式匹配中玩家人数，看到游戏中玩家人数；
- 查看游戏房间列表(按代码前缀分房间)，查看房间人数，支持创建房间、申请加入房间、退出房间等；
- 房间成员能够看到其他成员的ID、等级、信誉、准备、接受游戏、拒绝游戏等状态；
- 平台将能够记录玩家的恶意拒绝比赛行为，引入举报功能；

以上是VGReborn第一阶段要实现的。

如果有可能，能解密游戏对战的网络数据（需要大佬或官方技术支持），还能进一步获得这些能力：

- 获得玩家的对战数据，包括选择的英雄、战绩、游戏时间、游戏结果等；

如果能模拟数据发送到服务器，还能进一步获得这些能力：

- 1v1 SOLO模式，2v2模式，4v4模式，核心是通过模拟数据发送到服务器创建挂机玩家，比如，有两个玩家想要solo，可以通过模拟另外几个玩家加入匹配从而使游戏能够开始，游戏开始后模拟玩家是挂机状态。

## 技术实现

核心是通过加速器服务和MITM技术，使用此平台服务的玩家在游戏内数据将经过中间服务器，服务器将捕获数据并进行分析。

前端目前提供网页版。

后端通过Node.js开发。
